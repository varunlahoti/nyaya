# Indian Kanoon attribution logo

The Indian Kanoon API licence **requires** the "Powered by IKanoon" logo on top
of results (direct display) and in the About/footer (RAG use).

**Shipped:** `powered-by-ikanoon.svg` (desktop) and `powered-by-ikanoon-mobile.svg`
(mobile) reproduce the official mark (orange rings + blue "i" + "kanoon").
`components/Attribution.tsx` renders the right one per screen size.

**For strict brand compliance:** replace these two files with the exact official
assets from the IKanoon API **Terms & Conditions** page ("Attribution and logos"
table — desktop + mobile graphics), keeping the same filenames. Rules from the
licence: show the logo full and clearly visible; never alter, disproportionately
resize, or partially cover it; do not imply IKanoon endorsement of Nyaya.
